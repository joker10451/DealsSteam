# Справочник команд бота

## 🔑 Доступ к командам

### Для всех пользователей

**Основные команды:**
- `/start` — Начать работу с ботом, приветствие
- `/help` — Список всех команд
- `/tutorial` — Обучение работе с ботом

**Поиск и информация:**
- `/top` — Топ скидок сейчас
- `/find <название>` — Найти игру по названию
- `/price <название>` — Узнать цену игры
- `/stats` — Статистика канала

**Вишлист и уведомления:**
- `/wishlist` — Показать мой вишлист
- `/remove <название>` — Удалить игру из вишлиста
- `/cancel` — Интерактивное удаление из вишлиста
- `/genre <жанр>` — Подписаться на жанр
- `/genres` — Показать мои жанры
- `/free` — Подписаться на бесплатные игры
- `/free_off` — Отписаться от бесплатных игр

**Настройки уведомлений:**
- `/settings` — Настройки уведомлений
- `/notify_settings` — Алиас для /settings
- `/min_discount <процент>` — Минимальная скидка для уведомлений
- `/quiet_hours <начало> <конец>` — Тихие часы (без уведомлений)
- `/group_notify <on/off>` — Группировать уведомления

**Steam интеграция:**
- `/steam` — Подключить Steam аккаунт
- `/steamsync` — Синхронизировать вишлист Steam
- `/steamdisconnect` — Отключить Steam аккаунт
- `/steamstatus` — Статус подключения Steam
- `/freenotify` — Уведомления о бесплатных играх в Steam

**Геймификация (мини-игры):**
- `/score` — Мои баллы
- `/leaderboard` — Таблица лидеров
- `/games` — Список доступных игр
- `/profile` — Мой профиль
- `/achievements` — Мои достижения
- `/shop` — Магазин призов
- `/myrewards` — Мои купленные призы
- `/buy <id>` — Купить приз
- `/claim <id>` — Активировать приз (получить ключ)
- `/invite` — Реферальная ссылка

**Конкурсы:**
- `/giveaway` — Активные конкурсы
- `/mygiveaways` — Мои конкурсы

---

### Только для админов

**Публикация контента:**
- `/post` — Опубликовать скидки в канал
- `/testpost` — Тестовый пост в личку (не в канал)
- `/topday` — Опубликовать ТОП СКИДКУ ДНЯ (лучшая игра с максимальным score)
- `/gems` — Опубликовать скрытые жемчужины
- `/digest` — Опубликовать дайджест
- `/collection` — Опубликовать тематическую подборку
- `/testvk` — Тестовый пост в VK

**Статистика:**
- `/stats` — Статистика канала (расширенная для админа)
- `/channelstat` — Статистика канала
- `/keystats` — Статистика ключей

**Управление пользователями:**
- `/givekey <user_id> <ключ> <название>` — Выдать Steam ключ
- `/addpoints <user_id> <баллы>` — Начислить баллы

**Управление призами:**
- `/addkey <название> <ключ> <цена> [категория]` — Добавить ключ в магазин
- `/rewardstats` — Статистика призов

**Конкурсы:**
- `/creategiveaway` — Создать конкурс
- `/deletegiveaway <id>` — Удалить конкурс
- `/endgiveaway <id>` — Завершить конкурс досрочно
- `/giveawayhistory` — История конкурсов
- `/giveawaystat <id>` — Статистика участников конкурса
- `/kickgiveaway <giveaway_id> <user_id>` — Исключить участника
- `/cleantestgiveaways` — Очистить тестовые конкурсы
- `/vkgiveaway` — Создать конкурс в VK

**Другое:**
- `/broadcast <текст>` — Рассылка всем пользователям
- `/announcereferral` — Анонс реферальной программы
- `/tip` — Совет дня
- `/coop` — Кооперативные игры
- `/testgame` — Тестовая игра

## 📝 Примечания

1. **Проверка админа:** Все админские команды проверяют `user_id` через функцию `_admin_only()` в `handlers/admin.py`
2. **Админ ID:** Задаётся через переменную окружения `ADMIN_USER_ID` в `config.py`
3. **Тестовый пост:** Команда `/testpost` отправляет пост в личку админу, не публикует в канал
4. **Геймификация:** Все команды мини-игр доступны всем пользователям, но требуют инициализации БД
5. **VK публикация:** Команда `/testvk` для тестирования публикации в VK

---

## 🔧 Как добавить нового админа

1. Узнать `user_id` пользователя (можно через `/start` в логах)
2. Обновить переменную окружения `ADMIN_USER_ID` на Render.com
3. Перезапустить бот
