## 🔒 Проверка доступа

### Функция `_admin_only(message)`
Проверяет что `message.from_user.id == ADMIN_ID` из config.py

**Используется в:**
- `/post`
- `/gems`
- `/digest`
- `/collection`
- `/stats` (админская версия)
- `/givekey`
- `/addpoints`
- `/addkey`
- `/keystats`
- `/creategiveaway`
- `/endgiveaway`
- `/deletegiveaway`
- `/giveawayhistory`
- `/giveawaystat`
- `/testpost`
- `/broadcast`
- `/announce_referral`
- `/testgame`
- `/testvk`
- `/vkgiveaway`

### Где настраивается ADMIN_ID
`config.py`:
```python
ADMIN_ID = int(os.getenv("ADMIN_ID", "0"))
```

Переменная окружения `ADMIN_ID` на Render.com

## ⚠️ Важные замечания

### Команды без проверки доступа (публичные)
Все команды в:
- `handlers/wishlist.py` - вишлист
- `handlers/steam.py` - Steam интеграция
- `handlers/search.py` - поиск
- `handlers/notifications.py` - настройки
- `handlers/games.py` - геймификация
- `handlers/giveaway.py` - конкурсы (кроме админских)

### Команды с проверкой доступа (только админ)
Все команды в:
- `handlers/admin.py` - все команды проверяют `_admin_only()`

## 🔧 Как добавить новую команду

### Публичная команда
```python
@router.message(Command("mycommand"))
async def cmd_my_command(message: Message):
    await message.answer("Ответ")
```

### Админская команда
```python
@router.message(Command("mycommand"))
async def cmd_my_command(message: Message):
    if not _admin_only(message):
        await message.answer("⛔ Нет доступа.")
        return
    
    await message.answer("Ответ для админа")
```

### Регистрация
Команды регистрируются автоматически через `handlers/__init__.py`:
```python
def register_all(dp: Dispatcher):
    dp.include_router(admin.router)
    dp.include_router(search.router)
    # и т.д.
```
