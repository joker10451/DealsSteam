### 1. Улучшено форматирование постов

**До:**
```
🏎️ ВЫХОДНЫЕ-СКИДКИ · 14.03.2026
🎮 Car Mechanic Simulator 2021
━━━━━━━━━━━━━━
💸 880 руб  →  ✅ 158 руб  −82%
⭐️ Steam: 95%  ·  🏆 Крайне положительные
📝 Один из лучших в жанре. Брать не раздумывая.
```

**После:**
```
🔥 ОГОНЬ-СКИДКА · 14.03.2026
🎮 Car Mechanic Simulator 2021
━━━━━━━━━━━━━━
🔥 880 руб  →  ✅ 158 руб
🏷 Скидка: −82%
🏆 Steam: 95%  ·  Крайне положительные  ·  35 341 отзывов
💬 Скидка 82% на игру с высоким рейтингом. Однозначно брать.
```

**Изменения:**
- 🔥 Заголовок "ОГОНЬ-СКИДКА" для скидок 80%+
- 🔥 Эмодзи огня для цены при скидке 80%+
- 🏷 Отдельная строка для процента скидки
- 🏆 Динамические эмодзи для рейтинга (🏆 95%+, 👍 80%+, 🙂 70%+, 😐 <70%)
- 💬 Изменен эмодзи комментария с 📝 на 💬
- 📦 Индикатор для бандлов
- Ограничение похожих игр до 3 вместо всех
