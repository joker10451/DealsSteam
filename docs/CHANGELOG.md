# История изменений

## 14.03.2026 - Магазин призов 🆕

### Добавлено
- **Магазин призов**: пользователи могут обменивать баллы на реальные бонусы
- **9 типов призов**: подписки, Steam ключи, значки, персональные подборки
- **Система покупок**: автоматическое списание баллов и активация призов
- **Админ-панель**: команды для выдачи ключей и управления призами

### Призы в магазине
**Подписки:**
- ⚡️ Приоритетные уведомления (200 баллов, 7 дней)
- 💎 Расширенный вишлист до 50 игр (300 баллов, 30 дней)
- 🔥 Эксклюзивные скидки (500 баллов, 30 дней)

**Разовые призы:**
- 🎮 Steam ключи 5$/10$/20$ (1000/1800/3500 баллов)
- 🎯 Персональная подборка игр (150 баллов)

**Навсегда:**
- 👑 VIP значок (2000 баллов)
- 🏆 Значок легенды (5000 баллов)

### Новые команды
**Для пользователей:**
- `/shop` — магазин призов
- `/myrewards` — мои купленные призы
- `/buy [id]` — купить приз
- `/claim` — активировать приз (получить ключ)

**Для админа:**
- `/givekey [user_id] [ключ]` — выдать Steam ключ
- `/addpoints [user_id] [количество]` — начислить баллы
- `/rewardstats` — статистика по призам

### Технические изменения
- Создан модуль `rewards.py` с логикой магазина
- Добавлена таблица `user_rewards` в БД
- Интегрированы команды в `handlers/games.py` и `handlers/admin.py`
- Автоматические уведомления админу при покупке ключей

## 14.03.2026 - Система достижений

### Добавлено
- **Система достижений**: 15+ уникальных достижений с бонусными баллами
- **Отслеживание прогресса**: серии правильных ответов, ежедневная активность
- **Автоматическая разблокировка**: достижения проверяются после каждой игры
- **Уведомления**: игроки видят новые достижения сразу после разблокировки

### Типы достижений
- За баллы (от 1 до 1000 баллов)
- За количество игр (10, 50, 100 игр)
- За точность (80%, 90%, серии)
- За активность (ежедневные игры, лидерство)
- Специальные (челленджи, скриншоты)

### Новые команды
- `/achievements` — показать мои достижения

### Обновлённые команды
- `/profile` — теперь показывает серии, статистику достижений

### Технические изменения
- Создан модуль `achievements.py` с логикой достижений
- Добавлена таблица `user_achievements` в БД
- Расширена таблица `user_scores` (6 новых колонок)
- Интегрирована проверка достижений в `minigames.py`

## 14.03.2026 - Система мини-игр

### Добавлено
- **Система баллов**: пользователи зарабатывают баллы за участие в играх
- **Игра "Угадай игру по скриншоту"**: публикуется случайно после постов (20% шанс), награда 10 баллов
- **Таблица лидеров**: топ-10 игроков по баллам
- **Ежедневные челленджи**: новый челлендж каждый день, награда 50 баллов
- **Профиль пользователя**: статистика игр + активность в канале

### Новые команды
- `/games` — список доступных мини-игр
- `/score` — показать свои баллы и статистику
- `/leaderboard` — таблица лидеров
- `/profile` — полный профиль пользователя
- `/challenge` — показать челлендж дня

### Технические изменения
- Создан модуль `minigames.py` с логикой игр
- Создан модуль `handlers/games.py` с обработчиками команд
- Добавлены 5 новых таблиц в БД
- Интегрирована публикация игр в `publisher.py`
- Все 86 тестов проходят успешно

### База данных
Новые таблицы:
- `user_scores` — баллы и статистика пользователей
