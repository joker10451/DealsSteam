# 🎮 Telegram-бот, который ловит ВСЕ скидки на игры (+ геймификация!)

Привет, DTF! Решил поделиться проектом, который делал для себя, но думаю, многим пригодится.

## 🤔 Проблема

Пропускаешь раздачи в Epic, забываешь про распродажи в Steam, узнаёшь о скидках на игры из вишлиста постфактум. Существующие каналы либо спамят всем подряд (DLC за 30 рублей, никому не нужные инди), либо постят с задержкой, либо просто копируют RSS без фильтрации.

## ✨ Решение

Сделал автоматический канал в Telegram, который парсит Steam, GOG, Epic Games и CheapShark, фильтрует годноту и постит только то, что реально стоит внимания. **Но это не просто канал — это целая экосистема с геймификацией!**

---

## 🎯 Основные фичи

### 📢 Автоматика
- ✅ Проверяет магазины **3 раза в день**
- ✅ Фильтрует по рейтингам Steam (не постит шлак)
- ✅ Отмечает **исторические минимумы** цен
- ✅ Ловит **бесплатные раздачи** (Epic, GOG) — уведомления в реальном времени!
- ✅ Интеграция с **IGDB** для метаданных и обложек
- ✅ **AI-комментарии** о том, почему игра может быть интересна

### 👤 Персонализация
- 🔔 **Персональный вишлист** — добавляешь игру, бот пришлёт уведомление при скидке
- 🎮 **Импорт из Steam** — автоматически подтягивает твой вишлист и библиотеку
- 🔥 **Подписки на жанры** — получай уведомления только по интересным тебе жанрам
- 🌍 **Региональные цены** — сравнивай цены в разных странах Steam
- 🔔 **Уведомления о бесплатных играх** — не пропусти раздачи!

### 🎮 Геймификация (главная фишка!)

#### 🏆 Система баллов и достижений
- Играй в мини-игры и зарабатывай баллы
- Разблокируй **15+ достижений** (от "Первые шаги" до "Легенда")
- Каждое достижение даёт бонусные баллы
- Отслеживание серий правильных ответов и ежедневной активности

#### 🎯 Мини-игры
- **"Угадай игру по скриншоту"** — проверь свои знания игр
- **Ежедневные челленджи** — новое задание каждый день
- **Таблица лидеров** — соревнуйся с другими игроками
- **Система прогресса** — отслеживай свою статистику

#### 🏪 Магазин призов
Обменивай заработанные баллы на реальные призы:
- ⚡️ **Приоритетные уведомления** — получай скидки на 5 минут раньше всех
- 💎 **Расширенный вишлист** — увеличь лимит с 20 до 50 игр
- 🔥 **Эксклюзивные скидки** — доступ к закрытому каналу
- 🎮 **Steam ключи** — случайные игры на $5-20
- 💜 **Discord Nitro, Spotify Premium, Xbox Game Pass** — подписки на сервисы
- 👑 **VIP значки** — эксклюзивные статусы в профиле
- 🥇 **Эксклюзивные призы** для топ-игроков

#### 👥 Реферальная программа
- Приглашай друзей и получай **+100 баллов** за каждого
- Твой друг получает **+50 баллов** на старте
- Таблица лидеров по рефералам
- Уникальная реферальная ссылка для каждого

### 📊 Контент
- 📅 **Тематические дни**: RPG-понедельник, Экшен-вторник, Инди-четверг
- 📰 **Еженедельные дайджесты** с топ-10 скидок
- 💎 **Скрытые жемчужины** — инди с высокими рейтингами
- 🗳 **Голосование** за скидки (🔥 или 💩) — влияй на контент!

- Хештеги для удобного поиска

## 🛠 Технические детали (для интересующихся)

**Стек:**
- Python 3.11 + asyncio (полностью асинхронный)
- aiogram 3.x для Telegram API
- asyncpg для PostgreSQL (вишлисты, метрики, геймификация)
- APScheduler для расписания постов
- Парсеры для каждого магазина
- IGDB API для метаданных и обложек
- Pillow для генерации коллажей
- Steam Web API для интеграции с профилями

**Архитектура:**
- Модульная структура с разделением ответственности
- Единый пул соединений для БД
- Retry-логика для всех внешних API
- Система достижений с автоматической проверкой
- Магазин призов с поддержкой акций и лимитированных товаров
