# Документация постинга game-deals-bot

Описание всех типов постов, расписания, логики выбора скидок и вспомогательных систем.

---

## Расписание публикаций (МСК)

| Время       | Что публикуется                        |
|-------------|----------------------------------------|
| 08:00       | Авто-тест парсеров (только в лог)      |
| 09:00       | Скидка дня                             |
| 12:00       | Скидка дня + Скрытая жемчужина         |
| 14:00       | Скрытая жемчужина (hidden gem)         |
| 15:00       | Скидка дня                             |
| 18:00       | Скидка дня                             |
| 21:00       | Скидка дня                             |
| Вс 12:00    | Еженедельный дайджест                  |

Все времена задаются в `config.py` через `POST_TIMES = [(9,0),(12,0),(15,0),(18,0),(21,0)]` и `MSK = pytz.timezone("Europe/Moscow")`.

## Типы постов

### 1. Обычная скидка

Публикуется 5 раз в день по расписанию. Формат:

```
⚔️ RPG-ПОНЕДЕЛЬНИК · 01.01.2025

🎮 Название игры
━━━━━━━━━━━━━━
💸 ~~999 ₽~~  ➔  ✅ 299 ₽
🏷 Скидка: −70%
⏳ Скидка до: 15.01.2025
📊 Steam: 92%  👍 Очень положительные

📝 Высокий рейтинг — сообщество довольно. Хороший выбор.

#RPG #Экшен

[🛒 Открыть в Steam]  [🔥 0]  [💩 0]
```

После публикации платной скидки автоматически отправляется мини-игра «угадай цену».

### 2. Бесплатная игра

Когда `is_free == True` (Epic Games раздача или временно бесплатная в Steam):

```
🎁 БЕСПЛАТНО · 01.01.2025

🎁 Название игры
━━━━━━━━━━━━━━
💸 ~~Платная~~  →  🆓 БЕСПЛАТНО

📝 Бесплатно — просто берём, не думаем.

[🛒 Открыть в Epic Games]  [🔥 0]  [💩 0]
```

### 3. Исторический минимум

Когда `discount >= 70%` и CheapShark подтверждает исторический минимум:

```
⚡️ ИСТОРИЧЕСКИЙ МИНИМУМ · 01.01.2025

🎮 Название игры
━━━━━━━━━━━━━━
💸 ~~1999 ₽~~  ➔  ✅ 199 ₽
🏷 Скидка: −90%
...
```

### 4. Скрытая жемчужина (Hidden Gem)

Публикуется в 14:00 (и иногда в 12:00). Это малоизвестная инди-игра с высоким рейтингом Steam (≥85%) и большой скидкой. Источник — Steam Store API с фильтром по количеству отзывов (< 500) и рейтингу.

Формат аналогичен обычной скидке, но без темы дня.

### 5. Еженедельный дайджест

Публикуется каждое воскресенье в 12:00. Содержит:
- Топ-10 скидок недели по проценту скидки
- Топ-5 игр по голосам 🔥 за последние 7 дней

```
📅 ЛУЧШИЕ СКИДКИ НЕДЕЛИ — 05.01.2025

🏷 Топ по скидке:
1. 🎮 Игра А — −90%
2. 🟣 Игра Б — −85%
...

🔥 Топ по голосам:
1. Игра В — 42 🔥
...
```

### 6. Авто-тест парсеров

Запускается в 08:00 каждый день. Проверяет что все парсеры (Steam, GOG, Epic, CheapShark) возвращают хотя бы один результат. Результат пишется только в лог — в канал не публикуется.

## Логика выбора скидки для публикации

1. Собираются скидки из всех источников: Steam, GOG, Epic Games, CheapShark
2. Фильтрация по `MIN_DISCOUNT_PERCENT` (по умолчанию 50%)
3. Для Steam-скидок проверяется рейтинг: `MIN_STEAM_RATING` (по умолчанию 70%)
4. Дедупликация по названию (case-insensitive) — оставляется вариант с максимальной скидкой
5. Проверка `is_already_posted(deal_id)` — уже опубликованные пропускаются
6. Сортировка: сначала по `theme_score` (совпадение с темой дня), затем по `discount` по убыванию
7. Берётся первый подходящий deal, публикуется через `publish_single()`
8. `mark_as_posted(deal_id, ...)` сохраняет запись в БД

## Мини-игра «Угадай цену»

Запускается автоматически после публикации **платной** скидки (не бесплатной).

**Как работает:**
1. Из `deal.old_price` извлекается числовое значение в рублях
2. Генерируются 3 варианта-ловушки: случайные отклонения ±10–40% от правильной цены, округлённые до 10 ₽
3. Правильная цена сохраняется в таблицу `price_game` через `save_price_game(deal_id, price)`
4. В канал отправляется сообщение с 4 кнопками-вариантами (2×2)

**Обработка ответа:**
- Пользователь нажимает кнопку → `handle_price_game` callback
- Если `chosen == correct` → `✅ Правильно! Цена была N₽`
- Если нет → `❌ Неверно. Правильный ответ: N₽`
- После ответа кнопки убираются (`edit_reply_markup(None)`)
- Повторный ответ невозможен (кнопки уже убраны)

## Система вишлиста

Пользователи добавляют игры в личном чате с ботом.

**Как добавить:** написать название игры в личку боту (любой текст без `/`)

**Команды:**
- `/wishlist` — показать список
- `/remove [название]` — удалить запись

**Как работает матчинг:**
- При публикации скидки вызывается `get_wishlist_matches(deal.title)`
- Функция загружает все записи вишлиста и проверяет: `query in title.lower()` или `title.lower() in query`
- Всем совпавшим пользователям отправляется личное уведомление с ценой и кнопкой магазина

**Защита от дубликатов:** повторное добавление той же пары `(user_id, query)` возвращает `False` (UNIQUE constraint в БД).

## Система голосования 🔥 / 💩

Каждый пост содержит две кнопки голосования.

**Как голосовать:** нажать 🔥 (нравится) или 💩 (не нравится) под постом в канале.

**Защита от накрутки:** каждая пара `(deal_id, user_id)` уникальна в таблице `votes`. Повторное нажатие показывает «Ты уже голосовал!» без изменения счётчика.

**Обновление счётчиков:** после успешного голоса кнопки обновляются с актуальными числами через `edit_reply_markup`.

**Использование в дайджесте:** каждое воскресенье топ-5 игр по количеству 🔥 за последние 7 дней включается в еженедельный дайджест.

## Коллаж

К каждому посту прикрепляется изображение:
- Если доступны данные IGDB — собирается коллаж из обложки + до 3 скриншотов (модуль `collage.py`)
- Если коллаж не удался — используется одиночная обложка из IGDB или `deal.image_url`
- Если изображений нет — пост отправляется без фото

Коллаж: PNG 1280×400 пикселей, тайлы склеиваются горизонтально с тёмными разделителями.

## Региональные цены

Команда `/price [ссылка или название]` показывает цену игры в 5 регионах Steam: 🇷🇺 RU, 🇹🇷 TR, 🇦🇷 AR, 🇰🇿 KZ, 🇺🇸 US. Выделяется самый дешёвый регион.
