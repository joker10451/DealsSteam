# Руководство администратора

## Реферальная программа

### Как анонсировать

1. **Обнови закреплённое сообщение:**
   - Скопируй текст из `PINNED_MESSAGE.txt`
   - Закрепи в канале

2. **Опубликуй анонс:**
   - Скопируй текст из `REFERRAL_ANNOUNCEMENT.txt`
   - Опубликуй в канале
   - Можно добавить картинку/гиф для привлечения внимания

3. **Напомни через неделю:**
   - Создай пост с топ-3 рефереров
   - Покажи, сколько они заработали
   - Мотивируй других участвовать

### Команды для управления

**Просмотр статистики:**
```
/rewardstats — статистика по призам
/stats — общая статистика канала
```

**Работа с призами:**
```
/givekey [user_id] [ключ] — выдать Steam ключ
/addpoints [user_id] [баллы] — начислить баллы
```

**Пример выдачи ключа:**
1. Пользователь покупает ключ → тебе приходит уведомление
2. Покупаешь ключ на нужную сумму
3. Выдаёшь: `/givekey 123456789 XXXXX-XXXXX-XXXXX`
4. Пользователь получает ключ в личку

### Мониторинг роста

**Отслеживай метрики:**
- Количество новых пользователей в день
- Количество рефералов
- Конверсия в покупки призов
- Активность в мини-играх

**Оптимизация:**
- Если мало рефералов → увеличь награду или сделай акцию
- Если много покупок ключей → закупи ключи заранее
- Если низкая активность → добавь новые игры/призы

## Мини-игры

### Текущие игры

1. **Угадай цену** (автоматически после каждого поста)
   - Награда: 5 баллов
   - Работает в канале

2. **Угадай игру по скриншоту** (20% шанс после поста)
   - Награда: 10 баллов
   - Работает в канале

3. **Ежедневные челленджи** (нужно создавать вручную)
   - Награда: 50 баллов
   - Работает в боте

### Создание челленджа

Пока челленджи нужно создавать вручную через БД или добавить команду.
Планируется автоматическая генерация.

## Магазин призов

### Текущие призы

**Подписки:**
- ⚡️ Приоритетные уведомления — 200 баллов (7 дней)
- 💎 Расширенный вишлист — 300 баллов (30 дней)
- 🔥 Эксклюзивные скидки — 500 баллов (30 дней)

**Разовые:**
- 🎮 Steam ключ 5$ — 1000 баллов
- 🎮 Steam ключ 10$ — 1800 баллов
- 🎮 Steam ключ 20$ — 3500 баллов
- 🎯 Персональная подборка — 150 баллов

**Навсегда:**
- 👑 VIP значок — 2000 баллов
- 🏆 Значок легенды — 5000 баллов

### Добавление новых призов

Отредактируй `rewards.py` → `REWARDS_CATALOG`:

```python
"new_prize_id": {
    "name": "🎁 Название приза",
    "description": "Описание приза",
    "cost": 500,  # Цена в баллах
    "type": "one_time",  # или "subscription", "permanent"
    "duration_days": 30,  # Только для subscription
}
```

### Экономика призов

**Расчёт стоимости:**
- 1 правильный ответ = 5-10 баллов
- 1 день активной игры = ~30-50 баллов
- 1 неделя = ~200-350 баллов
- 1 месяц = ~800-1500 баллов

**Рекомендации:**
- Простые призы: 100-300 баллов (1-2 недели игры)
- Средние призы: 500-1000 баллов (2-4 недели)
- Дорогие призы: 1500-5000 баллов (1-3 месяца)

## Продвижение канала

### YouTube комментарии

Используй шаблоны из `YOUTUBE_COMMENTS.md`:
- Комментируй видео про игры
- Упоминай конкретные скидки
- Добавляй ссылку на канал
- Цель: 50-100 подписчиков в неделю

### Реферальная программа

- Мотивируй активных пользователей приглашать друзей
- Публикуй топ рефереров раз в неделю
- Делай акции (удвоенные баллы за рефералов)

### Контент

- Публикуй 5 раз в день (9:00, 12:00, 15:00, 18:00, 21:00 МСК)
- Еженедельный дайджест (воскресенье 12:00)
- Скрытые жемчужины (каждый день 14:00)

## Техническая поддержка

### Логи и мониторинг

Проверяй логи на Render.com:
- Ошибки парсеров
- Проблемы с БД
- Ошибки публикации

### Резервное копирование

База данных на Supabase автоматически делает бэкапы.
Код на GitHub — всегда можно откатиться.

### Обновления

Все изменения автоматически деплоятся при push в master:
```bash
git add -A
git commit -m "Описание изменений"
git push origin master
```

Render автоматически подхватит изменения через 2-3 минуты.
