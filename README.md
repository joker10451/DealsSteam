# 🎮 GameDealsBot - Telegram бот для отслеживания скидок на игры

Автоматический бот для мониторинга и публикации скидок на игры из Steam, GOG, Epic Games и других магазинов. Публикует в Telegram канал и VK группу с умной фильтрацией, персональными уведомлениями и геймификацией.

## 📋 Основные возможности

### 🔍 Мониторинг скидок
- **Автоматический парсинг** скидок из Steam, GOG, Epic Games, CheapShark
- **Умная фильтрация** по рейтингу, скидке, цене, жанрам
- **Детекция ошибок цен** - находит аномальные скидки (90%+)
- **Исторические минимумы** через IsThereAnyDeal API
- **Публикация 3 раза в день** по расписанию (10:00, 15:00, 20:00 МСК)

### 📢 Публикация контента
- **Telegram канал** - основная площадка с интерактивными кнопками
- **VK группа** - автоматическое дублирование постов
- **Эмоциональные заголовки** - "ОГОНЬ! Скидка 80%" вместо сухих описаний
- **Короткие описания** - суть игры в 1-2 предложениях
- **Комментарии бота** - "Стоит ли брать?" для каждой игры
- **Еженедельные дайджесты** - топ скидок недели (воскресенье 12:00)

### 👤 Персонализация
- **Вишлист** - добавляй игры и получай уведомления о скидках
- **Подписка на жанры** - уведомления по любимым жанрам
- **Настройки уведомлений**:
  - Минимальная скидка (по умолчанию 30%)
  - Тихие часы (не беспокоить ночью)
  - Группировка уведомлений
- **Интеграция со Steam**:
  - Импорт вишлиста (до 100 игр)
  - Импорт библиотеки (автоматическое исключение купленных игр)
  - Поддержка приватных профилей

### 🎯 Интерактив
- **Голосование** - 🔥 Огонь / 💩 Мимо для каждой скидки
- **Быстрое добавление в вишлист** - кнопка ➕ в каждом посте
- **Поделиться с другом** - персональная ссылка для каждой игры
- **Поиск по тегам** - команда `/find` для поиска скидок

### 🎁 Геймификация (временно отключено)
- ⏸ **Мини-игры** - будут включены при достижении 500+ подписчиков
- ⏸ **Система достижений** - в разработке
- ⏸ **Магазин призов** - планируется
- ⏸ **Реферальная программа** - будет добавлена позже

**Почему отключено:** Геймификация работает только с активной аудиторией. Сейчас фокус на качестве контента и росте подписчиков.

### 🎉 Конкурсы
- **Розыгрыши Steam ключей** - честная система с commit-reveal
- **Бонусы за рефералов** - больше друзей = больше шансов
- **Автоматический выбор победителей** - прозрачный алгоритм
- **Напоминания** - уведомления за 1 час до окончания

### 📊 Аналитика
- **Метрики вовлеченности** - показы, голоса, клики, добавления в вишлист
- **CTR вишлиста** - эффективность постов
- **Топ-5 постов** - самые популярные скидки
- **Статистика пользователей** - активность, баллы, достижения

## 🎯 Умная фильтрация

### Критерии публикации
1. **Минимальная скидка:** 30% (настраивается)
2. **Минимальный рейтинг Steam:** 70% (для платных игр)
3. **Минимальная цена:** 50₽ (настраивается)
4. **Фильтр бандлов:** опционально
5. **Фильтр 18+:** опционально

### Приоритеты публикации
1. **Критические ошибки цен** (90%+ скидка)
2. **Бесплатные игры**
3. **Исторические минимумы**
4. **Высокие скидки** (80%+)
5. **Тема дня** (жанр дня меняется ежедневно)
6. **Высокий рейтинг** (95%+)

### Дедупликация
- По IGDB ID (одна игра = один пост)
- По названию (нормализация)
- Проверка уже опубликованных

## 🔒 Безопасность

### Конкурсы
- **Commit-reveal схема** - seed генерируется до старта, публикуется SHA-256
- **Проверяемость** - после розыгрыша seed раскрывается для проверки
- **Честность** - невозможно подтасовать результаты

### Данные пользователей
- **Минимум PII** - храним только Telegram ID
- **Опциональная интеграция** - Steam привязка по желанию
- **Приватность** - поддержка приватных Steam профилей

### API ключи
- Все токены в переменных окружения
- Никаких хардкодов в коде
- `.env` в `.gitignore`
