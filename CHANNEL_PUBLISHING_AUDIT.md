# 🔍 Аудит публикаций в канал

## ✅ Что публикуется в канал (РАЗРЕШЕНО)

### Основные посты (по расписанию):
1. **Скидки** - 5 раз в день (9:00, 12:00, 15:00, 18:00, 21:00)
   - Функция: `publish_single()` в `publisher.py`
   - Триггер: `check_and_post()` в `scheduler.py`

2. **Скрытые жемчужины** - 1 раз в день (14:00)
   - Функция: `post_hidden_gems()` в `scheduler.py`

3. **Еженедельный дайджест** - воскресенье 12:00
   - Функция: `post_weekly_digest()` в `scheduler.py`

4. **Кооп-дайджест** - пятница 18:00
   - Функция: `post_coop_digest()` в `scheduler.py`

5. **Тематические подборки** - выходные
   - Суббота 11:00: кооперативы
   - Воскресенье 15:00: бюджетные игры
   - Функция: `post_themed_collection()` в `themed_collections.py`

### Дополнительные посты (по событиям):
6. **Бесплатные игры** - при обнаружении
   - Функция: `publish_free_game()` в `free_game_monitor.py`
   - Триггер: `check_epic_free_games()` каждые 2 часа

7. **Конкурсы** - при создании админом
   - Функция: `publish_giveaway()` в `giveaways.py`
   - Триггер: команда `/creategiveaway`

## 🔒 Гарантии

1. **Только 1 пост за раз** - все функции публикации используют `send_with_retry()` с защитой от flood
2. **Нет спама ошибками** - все уведомления об ошибках отключены или идут только админу
3. **Нет мини-игр** - все игровые функции закомментированы
4. **Четкое расписание** - все публикации по cron, никаких случайных постов

## 🛠 Как изменить расписание

### Изменить время публикаций скидок:
Установи переменную окружения `POST_TIMES`:
```bash
POST_TIMES="12:0,18:0"  # Только 2 раза в день: 12:00 и 18:00
```

### Отключить скрытые жемчужины:
Закомментируй в `bot.py` строку 169:
```python
# scheduler.add_job(post_hidden_gems, CronTrigger(hour=14, minute=0, timezone=MSK), name="hidden_gems")
```

### Отключить тематические подборки:
Закомментируй в `bot.py` строки 250-262:
```python
# scheduler.add_job(lambda: post_themed_collection("weekend_coop"), ...)
# scheduler.add_job(lambda: post_themed_collection("budget_games"), ...)
```

## ⚠️ Важно

Если видишь лишние посты в канале:
1. Проверь логи бота: `docker logs <container_id>`
2. Проверь что ADMIN_ID настроен правильно в `.env`
3. Убедись что нет ручных команд `/post`, `/gems` и т.д.
4. Проверь что мини-игры действительно отключены (см. этот файл)
