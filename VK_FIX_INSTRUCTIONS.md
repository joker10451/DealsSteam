# 🔧 Исправление VK публикации

## Проблема

VK публикация работает, но **без картинок**. Ошибка:
```
VK API error in photos.getWallUploadServer: 
error_code 27 - Group authorization failed: method is unavailable with group auth
```

## Причина

Используется **групповой токен** (сервисный ключ группы), но метод `photos.getWallUploadServer` недоступен для групповых токенов. Нужен **пользовательский токен**.

## Решение

### Вариант 1: Пользовательский токен (РЕКОМЕНДУЕТСЯ)

1. Зайди на https://vkhost.github.io/
2. Выбери **VK Admin** (или создай Standalone приложение)
3. Получи токен с правами: `photos, wall, offline, groups`
4. Замени `VK_ACCESS_TOKEN` на Render.com на этот токен

### Вариант 2: Публикация без фото

Если не хочешь возиться с токенами, можно публиковать без картинок. Для этого:

1. Открой `game-deals-bot/vk_publisher.py`
2. Найди строку 120:
```python
if not VK_ENABLED or not VK_TOKEN or not VK_GROUP_ID:
```

3. Измени логику загрузки фото - вместо ошибки просто пропускай фото:

```python
# В функции post_deal_to_vk, строка ~180
# Вместо:
if image_url:
    attachment = await _upload_photo(image_url)
    if attachment:
        attachments.append(attachment)

# Сделай:
if image_url:
    attachment = await _upload_photo(image_url)
    if attachment:
        attachments.append(attachment)
    else:
        # Публикуем без фото, используем ссылку на Steam
        attachments.append(deal.link)
```
