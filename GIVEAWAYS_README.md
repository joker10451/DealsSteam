# 🎁 Система конкурсов (Giveaways)

Полнофункциональная система розыгрышей для привлечения и удержания подписчиков канала.

## ⚡ Быстрый старт

### Создать конкурс

```bash
/creategiveaway Portal 2 | Классика от Valve | steam_key | XXXXX-XXXXX-XXXXX | 72
```

### Завершить досрочно

```bash
/endgiveaway giveaway_1234567890
```

## 📋 Возможности

- ✅ Розыгрыш Steam ключей, баллов, подписок
- ✅ Автоматическая проверка подписки на канал
- ✅ Защита от мультиаккаунтов (минимальный возраст)
- ✅ Автоматическое завершение по таймеру
- ✅ Случайный выбор победителя
- ✅ Автоматическая выдача призов
- ✅ Объявление результатов в канале
- ✅ Счётчик участников в реальном времени

## 🎯 Типы призов

| Тип | Описание | Пример значения |
|-----|----------|-----------------|
| `steam_key` | Steam ключ игры | `XXXXX-XXXXX-XXXXX` |
| `points` | Баллы для магазина | `1000` |
| `subscription` | Приз из магазина | `vip_badge` |

## 📊 Архитектура

```
giveaways.py              # Основная логика
handlers/giveaway.py      # Команды и callback'и
database.py               # Таблицы giveaways, giveaway_participants
bot.py                    # Регистрация + планировщик
```

## 🔄 Жизненный цикл конкурса

1. **Создание** → `/creategiveaway` → запись в БД + пост в канале
2. **Участие** → кнопка "Участвовать" → проверки + добавление в БД
3. **Завершение** → таймер/команда → выбор победителя + выдача приза
4. **Объявление** → обновление поста + уведомление победителя

## 🛡️ Защита

- Проверка подписки на канал (опционально)
- Минимальный возраст аккаунта (по умолчанию 7 дней)
- Уникальность участия (один раз на конкурс)
- Атомарная выдача ключей (FOR UPDATE SKIP LOCKED)

## 📈 Стратегия роста

### "Троянский конь"

1. Купите ключ популярной игры (100-200₽)
2. Создайте конкурс на 2-3 дня
3. Поделитесь в 5-10 геймерских чатах
4. Получите резкий рост подписчиков

**Пример сообщения:**
```
Парни, тут Portal 2 разыгрывают!
Подписка = участие, розыгрыш через 3 дня
[ссылка на канал]
```

## 🔧 Конфигурация

Все настройки через параметры функций:
- `duration_hours` — длительность конкурса
- `require_channel_sub` — требовать подписку (default: True)
- `min_account_age_days` — минимальный возраст (default: 7)

## 🚀 Автоматизация

Планировщик проверяет завершённые конкурсы каждые 15 минут:
```python
scheduler.add_job(
    check_ended_giveaways,
    "interval",
    minutes=15,
    name="check_giveaways"
)
```

## 💡 Советы

- Проводите конкурсы **регулярно** (раз в 1-2 недели)
- Чередуйте **типы призов** (ключи, баллы, статусы)
- Анонсируйте **заранее** для максимального охвата
- Отслеживайте **метрики** (участники, прирост, retention)
- Делитесь в **тематических чатах** для виральности

## ⚠️ Важно

- Всегда выдавайте призы победителям (репутация!)
- Проверяйте ключи перед созданием конкурса
- Не злоупотребляйте — качество > количество
- Следите за retention после конкурсов
